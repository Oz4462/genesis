# GENESIS R&D System Contract (per GENESIS_PLATFORM_BUILD_TODO §A2)

**Flow (the one true chain):**

```
Dream / Frage
→ MoonshotIntent + clarification
→ Prior-Art + DevelopmentFrontMap (scout/scholar/skeptic + wissensbasis)
→ CapabilityGap + MilestoneLadder
→ Hypothesis + ExperimentPlan (experiment_designer)
→ Teststand / PrototypeSpec (teststand_architect + cad)
→ SimulationSpec + Runner + Receipt (simulation + physics_validation)
→ MeasurementRun (reality δ⁺ + bench)
→ LearningDelta (lernmaschine 8-step + apply)
→ ProofPackage + next Readiness level
```

**Data Models (minimum):**
- Hypothesis (measurable prediction, falsifiability)
- ExperimentPlan (variables, measures, safety, abort)
- MeasurementRun (data + provenance + uncertainty)
- LearningDelta (what changed, evidence, apply targets)

**Gates:**
- No hypothesis without measurable + falsifiable.
- No experiment without safety/abort + measure.
- No learning update without before/after evidence.
